# ATAS 定时下单策略

在图表上添加策略后，可在参数中配置预定时间、买卖方向、市价/挂单及数量，到达指定时间后自动下单。

## 参数说明

| 分组 | 参数 | 说明 |
|------|------|------|
| 定时设置 | 预定小时/分钟/秒 | 当日触发时间（本地时间） |
| 订单设置 | 交易方向 | 买入 / 卖出 |
| 订单设置 | 订单类型 | 市价单 / 挂单(限价) |
| 订单设置 | 限价 | 挂单模式下必填 |
| 订单设置 | 交易数量 | 下单手数 |

## 行为说明

- 策略启动后，到达预定时间时触发一次下单。
- 若启动时预定时间**已过**，则视为**次日**同一时刻下单（不会立即补发）。
- 每个交易日触发一次，次日自动重置后可再次触发。
- 日志文件：`D:\ATASData\ScheduledOrderStrategy.log`

## 编译

```powershell
dotnet build ATASScheduledOrder.sln -c Release
```

Release 编译后会自动复制到：

`C:\Users\Administrator\AppData\Roaming\ATAS\Strategies\`

## 使用

1. 在 ATAS 图表上添加策略「定时下单策略」。
2. 选择账户（Portfolio）与合约。
3. 配置预定时间与订单参数。
4. 启动策略并等待到达预定时间。
